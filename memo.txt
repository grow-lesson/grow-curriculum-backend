環境構築(Windows)
参考記事
https://tech-popeye.com/ruby-on-rails-mysql-windows/

mysql install
https://downloads.mysql.com/archives/c-c/

ruby install(Ruby+Devkit 3.2.2-1 (x64) )
https://rubyinstaller.org/downloads/

手順
1. rails new app_name -d mysql

2. bundle install

3. Gemfileの40行目を
gem 'tzinfo-data'
gem 'tzinfo-data', platforms: [:mingw, :mswin, :x64_mingw]
これに変える。

4. bundle install

5. database.ymlにusernameとpasswordを設定

6. rails db:create

7. rails db:migrate

8. rails cでカラムを追加するときは
   User.create(name: 'grow' , email: 'example@gmail.com' , password: "admin" , hobbies: 'Soccer' , languages: 'Ruby' , bio: 'よろしくお願いします。' )
　このようにemailとpassword両方必須である。
